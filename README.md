# Ejercicio useState y useEffect

## Uso:
- Descargar el código:
```bash
git clone https://github.com/tonikny/ra-m2-usestate-useeffect.git
```
- Cambiar al directorio de la aplicación:
```bash
cd ra-m2-usestate-useeffect
```
- Instalar dependencias:
```bash
npm install
```
- Ejecutar:
```bash
npm run dev
```
